# ウェブデザイン技能検定 3級 傾向と対策クイズ

ウェブデザイン技能検定3級（学科）対策のための学習用クイズアプリです。  
ランダム出題・過去問対応・復習機能により、効率よく理解を深めることができます。

---

## 特徴

- ✅ ランダム25問出題（本番形式）
- ✅ 過去問（05・06）対応
- ✅ 正解率表示（合格目安：70%以上）
- ✅ 復習モード（問題・選択肢・解説付き）
- ✅ レスポンシブ対応（スマホOK）

---

## 復習機能（重要）

回答後に以下を確認できます：

- 問題文
- 選択肢（どれを選んだか）
- 正解
- 解説

 「なぜ間違えたか」が分かる設計

---

## 画面構成

### トップページ
- ランダムスタート（最新問題）
- 過去問スタート

### クイズ画面
- 25問出題
- 正解 / 不正解表示
- 解説モーダル

### 結果ページ
- 今回のスコア
- ベストスコア
- 復習（今回の問題）
